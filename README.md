# DoodleChat

Real-time collaborative doodle program created to explore real-time multi-user canvas manipulation and websockets.
Made in Fall 2025 for SUNY Polytechnic Undergraduate Degree in Computer Science's Capstone Project.

## Built with:
- [Node.js](https://nodejs.org/en)
- [Express.js](https://expressjs.com/)
- [Socket.io](https://socket.io/)
- [P5.js](https://p5js.org/)
- [Font Awesome (for UI icons)](https://fontawesome.com/)
